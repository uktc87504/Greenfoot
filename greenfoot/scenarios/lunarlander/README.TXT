PROJECT TITLE:
Lunar Lander

PURPOSE OF PROJECT: This is an example of a simple game. We wrote this example
to demonstrate how to handle key input: the rocket thrust can be controlled by
pressing the "down" key on the keyboard.

VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
Compile all class and press the run button.
